# Bitlog ( Simple Blog Application)

This is a simple **Blog Application** built using **HTML, CSS, JavaScript, PHP, and MySQL**.  
It allows users to create, edit, and delete blog posts, with images and author details.  
The website has a dark theme and a clean, modern design.

---

## Features

- Create, edit, and delete blog posts  
- Add a blog image (shown as a thumbnail)  
- View all posts on the home page  
- View each post in detail  
- Responsive dark-themed UI  
- MySQL database to store posts and users  
- Ready to host online

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL
